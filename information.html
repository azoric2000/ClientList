User instructions

This application provides you the most accurate information on our Account Center clients. This app was designed to provide you with the information you need in the most easily consumable format possible.

How to use this app:

  • Environment Selection

    • The Environment list connects you to the instance of Account Center at different stages of development. The environments listed are:
      • DEV The Account Center Development environment. This is where developers implement new code and test it.
      • SIT System Integration Testing environment. At this stage all developers integrate code together.
      • UAT (coming soon) User Integration Testing. This is the environment where most testing occurs to. Content and branding changes are mostly implemented here. This is also an environment available to clients for their testing as well
      • CERT (coming soon) Production. What you see here is exactly as Account Center is set up for our clients and customers.

  • From the Client tab

    • Select a client from the list on the left.
    • If you cannot find a client quickly, use the search field at the top of the left column.
    • The list will filter as you type.
    • Reset the list by clicking the [ X ] button next to the search bar.
    • Click the checkbox next to the client name to see all available information.
      • Client information box will display in the main column.
      • Each client is separated by a colored vertical bar on the left.
      • Remove the client information box by clicking the blue minus "-" icon in it's upper right.
      • Client information is listed in a grid format with Yes/No features represented by a Check or X icon.
    • Click the blue reset button at the top of the main column to clear all selected clients.

  • From the Feature tab

    • Select a feature from the list on the left.
    • That feature will appear at the top of the main section with all clients utilizing that feature listed below.

If you are experience a problem or you have a feature request for this applicaiton please contact a system administrator

Enjoy!
