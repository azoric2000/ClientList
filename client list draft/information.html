User instructions

The data here is provided from our CERT environment, not Production. This data is believed to be the best effort to privide a view into production as possible.

The CERT environment is backloaded from production every Wednesday.

How to use this app

  • From the Client tab
    • Select a client from the list on the left.
    • All available information on that client will display in the large area
    • Clicking a link in the Client information grid wil provide you with a list of all clients with that feature
  • From the Feature Tab
    • Select a feature from the list on the left
    • That feature will appear at the top of the main section
    • Click the refresh button to see all clients with the selected features
